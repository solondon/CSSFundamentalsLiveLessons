Have you heard about Boks?

May 2, 2009 3 Comments

Have you heard about Boks? You would, if you followed Blueprint news. Boks is a visual layout editor, much like Construct, but built with Adobe AIR so it's a desktop application that can save, load & export projects. It also has a lot more features than Construct, like supporting append/prepend classes and allowing custom HTML content in columns. If you ask me, this is the best realization of a fully WYSIWYG semantic editor so far, and it's only at version 0.1.8! Definitely check it out, and send your feedback to the developer, Quentin T.
