An Introduction to CSS Syntax

The basics

Stylesheets are made up of rules, which comprise selectors and property:value pairs. A typical CSS rule might be:


body { 
	color:green;
}

A quick search for "CSS 2.1 Selectors" will return the official document from the W3C describing all the possibilities for selectors. The important takeaways are:

  • Selectors can include tag names, IDs, classes, and a variety of other options with varying browser support
  • Selectors can be grouped with commas to combine rulesets
  • Two or more selectors separated by whitespace are descendant selectors
  • The asterisk is a wildcard that matches any element
  • IDs are denoted using the hash symbol: #myid
  • Classes are denoted by periods: .myclass
  • IDs and classes can be combined with tag names to target more specific cases.

A complex example


ul#takeaways li.ids * {
	color:blue;
}
